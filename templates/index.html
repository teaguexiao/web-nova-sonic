Tool Information

Nova Sonic 2 Now Available!

This demo page has been upgraded to support Amazon Nova Sonic 2, offering the following core features:

  • Low-Latency Real-Time Conversations: Experience smooth, real-time voice interaction
  • Multilingual Speech Understanding: Support for speech recognition across multiple languages and speaking styles
  • 22 Expressive Voices: Including male and female voices in English, Spanish, German, French, Italian, Portuguese, and Hindi
  • Natural Conversational Processing: Understands and adapts to pauses, hesitations, and interruptions in user speech while maintaining conversation context throughout
  • Cross-Modal Input: Support for text messages in the same session without switching context
  • Tool-Calling Capabilities: Provides precise responses based on enterprise-specific data and executes intelligent workflows (e.g., booking services)
  • Asynchronous Task Processing: Users can continue conversations while tools execute operations in the background
